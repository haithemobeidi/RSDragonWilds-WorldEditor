Actions

Copy Buildings INTO This World
Pick a donor world to copy buildings from. All of the donor's walls, floors, and roofs will be added to this world, preserving its existing state. Pieces appear at their original coordinates (worlds share the same map seed).
⚠️ Only passive building pieces (walls/floors/roofs) actually render. Chests and crafting stations will be added to the file but won't appear in-game (Category 2 limitation). Pre-existing pieces in this world that spatially overlap the donor's pieces may be lost.
Clone World
Create a complete duplicate of this world with a new name and identity. The copy is a separate save with its own WorldSaveGuid — both worlds can coexist. Useful for branching saves or backups.
Set World Mode
Toggle between Standard (default difficulty) and Custom (with custom difficulty settings). Custom mode unlocks editing of game balance values like resource costs, drop rates, and progression scaling.
⚠️ Converting Standard → Custom may break locked chests/totems (Bug #001). The chests still exist but their owner reference becomes stale. Affected items can be re-locked manually in-game.
